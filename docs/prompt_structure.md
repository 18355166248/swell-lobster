# SwellLobster 提示词结构文档

> 最后更新: 2026-03-12

本文档全面梳理 SwellLobster 项目的提示词系统架构，包括静态提示词文件、动态生成逻辑、以及各组件的协作关系。

---

> **框线错位原因**：等宽字体下中文占 2 个显示宽度（全角），英文与符号占 1 个（半角），同一行字符数相同但视觉宽度不同，框线就会错位。**保证对齐**：仅使用半角字符（见下纯英文版），或在支持「CJK 等宽」的字体下查看。
