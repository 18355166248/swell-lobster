## 如何本地跑通

1. 安装 Python 依赖：pip install -e .
2. 启动后端：swell-lobster serve
3. 启动前端：cd apps/web-ui && npm install && npm run dev
4. 浏览器打开前端地址，侧栏可切换各菜单；后端未启动时 Topbar 显示「未连接」，各页请求会报错，属预期。
