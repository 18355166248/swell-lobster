# Tasks — 指南

## 目录职责

这个目录用于存放阶段性任务实例文档。

- 大任务说明
- 功能设计草案
- 重构执行单
- 需要保留审计轨迹的专项任务记录

## 规则

- 文件名建议使用 `YYYY-MM-DD-任务名.md`
- 新任务优先从 [TEMPLATE.md](TEMPLATE.md) 复制
- 这里的文档默认是阶段性工件，不直接等同于长期规则
- 如果某份任务文档中的约束已经稳定，应把它提炼到 `AGENTS.md`、README 或 `docs/` 下的长期文档
